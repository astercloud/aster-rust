# GitHub 模块

GitHub 集成模块，提供 GitHub Actions 工作流设置、PR 管理等功能。

## 功能概述

- **工作流管理**: 设置 GitHub Actions 工作流
- **PR 管理**: 获取 PR 信息、评论、创建 PR
- **CLI 检查**: 检查 GitHub CLI 安装和认证状态

## 文件索引

| 文件 | 说明 |
|------|------|
| `mod.rs` | 模块入口 |
| `workflow.rs` | GitHub Actions 工作流管理 |
| `pr.rs` | PR 信息获取、评论、创建 |
